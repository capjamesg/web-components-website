Installation

Install darkmode.js by downloading the darkmode.min.js file from the darkmode.js repository and linking to it from the <head> of your HTML document.

                
<script src="darkmode.min.js"></script>
<style id="darkmode" media="all">
</style>

Then, enclose any elements you want to be affected by dark mode in a <span> element with the class seasonal_emoji, like so:

                
<span class="seasonal_emoji">üßá</span>
