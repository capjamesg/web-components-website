Web Components and Utilities 🛠️

A collection of HTML Components and JavaScript utilities for use in web sites.

JavaScript Utilities

  • seasonal.js🌲

    Change emojis on your website for seasonal events.

  • darkmode.js 🌙

    A JavaScript library that allows you to toggle dark mode on a page.

  • highlight.js 📝

    A JavaScript library that allows you to highlight text on a page.

  • hovercard.js 📇

    Show link previews when hovering over links.

  • fragmention.js 📇

    Support fragmention links in your web site.

Web Components

  • RandomText 📝

    A web component that displays a random string from a list of sentences.

  • ProfileCard 📇

    A web component that displays a profile card with an avatar and name.
